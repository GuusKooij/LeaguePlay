LeaguePlay.exe is the main file. Help button in program describes the functions.

If you want to use the 'Generate Playstyle' function, import LeaguePlayPlaystyles.json. into your League of Legends client. Go to Collection -> Items -> Import Item Sets -> Select LeaguePlayPlaystyles.json.
